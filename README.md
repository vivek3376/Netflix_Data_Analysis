# Netflix Data Analysis using SQL & Power BI

## Project Overview

This project analyzes Netflix's content library using Python, MySQL, SQL, and Microsoft Power BI. The objective is to clean the data, perform SQL analysis, and create an interactive dashboard that provides meaningful business insights.

---

## Objectives

* Analyze Netflix Movies and TV Shows
* Perform data cleaning and preprocessing
* Execute SQL queries for business insights
* Build an interactive Power BI dashboard
* Present key findings through visualizations

---

## Tools & Technologies

* Python
* Pandas
* MySQL
* SQL
* Microsoft Power BI
* DAX
* Git & GitHub

---

## Dataset

* Source: Kaggle (Netflix Titles Dataset)
* Records: ~8,800
* Features: 12

The dataset contains information about titles, type, country, directors, ratings, genres, release year, duration, and descriptions.

---

### Filters

* Content Type
* Country
* Release Year

---

## Key Insights

* Movies make up the majority of Netflix content.
* The United States contributes the highest number of titles.
* Drama and International Movies are among the most common genres.
* Netflix content increased significantly after 2015.
